Course Syllabus

Module 1: Introduction to Figma

  • Overview of Figma as a collaborative design tool
  • Understanding the interface and navigation
  • Creating an account and setting up preferences
  • Exploring the different views: Design, Prototype, and Code
  • Overview of basic tools and functionalities

Module 2: Design Basics in Figma

  • Introduction to design principles and best practices
  • Working with shapes, text, and images
  • Understanding layers and organization
  • Utilizing grids and layouts for consistency
  • Introduction to styles and components

Module 3: Collaboration and Sharing

  • Collaborating in real-time with team members
  • Using comments and annotations for feedback
  • Version control and design history
  • Sharing designs with stakeholders
  • Permissions and access control

Module 4: Advanced Design Techniques

  • Responsive designs with constraints
  • Vector editing and advanced manipulation
  • Using plugins to extend capabilities
  • Integrating third-party tools
  • Advanced prototyping and animations

Module 5: Design Systems & Component Libraries

  • Introduction to design systems
  • Managing design systems in Figma
  • Building component libraries
  • Design tokens and variables
  • Integrating systems into development

Module 6: Handoff & Developer Collaboration

  • Generating design specs and assets
  • Inspecting designs and exporting assets
  • Developer handoff tools and plugins
  • Bridging design and dev teams
  • Best practices for collaboration

Module 7: Workflow Optimization & Tips

  • Keyboard shortcuts and productivity hacks
  • Automating tasks with plugins
  • Optimizing workflow efficiency
  • Troubleshooting common issues
  • Staying updated with new features

Module 8: Final Project & Portfolio Review

  • Applying skills in a final design project
  • Project presentation and feedback
  • Building a portfolio of Figma projects
  • Job hunting and career guidance
